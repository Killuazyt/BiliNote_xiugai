# BiliNote 安装指南

## 项目简介
BiliNote 是一个开源的 AI 视频笔记助手，支持通过哔哩哔哩、YouTube、抖音等视频链接，自动提取内容并生成结构清晰的 Markdown 格式笔记。

---

## 环境要求

| 依赖 | 版本 |
|------|------|
| Python | 3.11.1 |
| pip | 26.0.1 |
| Node.js | 22.13.1 |
| pnpm | 10.28.2 |
| ffmpeg | 6.1.1 |

---

## 安装步骤

### 步骤 1: 克隆项目仓库 ✅

```bash
git clone https://github.com/JefferyHcool/BiliNote.git
cd BiliNote
cp .env.example .env
```

**状态**: 已完成

---

### 步骤 2: 安装 ffmpeg 依赖 ✅

ffmpeg 用于音频处理与转码，是必须的依赖。

```bash
# Ubuntu / Debian
sudo apt update && sudo apt install -y ffmpeg
```

**状态**: 已完成 - ffmpeg version 6.1.1-3ubuntu5

---

### 步骤 3: 安装后端依赖 (FastAPI) ✅

```bash
cd /workspace/BiliNote/backend
pip install -r requirements.txt
```

**状态**: 已完成

主要依赖包括：
- fastapi==0.115.12
- faster-whisper==1.1.1
- openai==1.70.0
- uvicorn==0.34.0
- yt-dlp==2025.3.31

---

### 步骤 4: 安装前端依赖 (Vite + React) ✅

```bash
cd /workspace/BiliNote/BillNote_frontend
pnpm install
```

**状态**: 已完成

主要依赖包括：
- react==19.2.4
- vite==6.4.1
- antd==5.29.3
- tailwindcss==4.2.1

---

## 启动项目

### 启动后端

```bash
cd /workspace/BiliNote/backend
python main.py
```

后端服务将在 `http://127.0.0.1:8483` 运行

### 启动前端

```bash
cd /workspace/BiliNote/BillNote_frontend
pnpm dev
```

前端服务将在 `http://localhost:5173` 运行

## 快速启动脚本

项目提供了快速启动脚本：

### 启动服务

```bash
cd /workspace/BiliNote

# 同时启动前后端
./start.sh

# 仅启动后端
./start.sh --backend

# 仅启动前端
./start.sh --frontend

# 查看状态
./start.sh --status
```

### 停止服务

```bash
./stop.sh
```

### 日志文件

- 后端日志: `/tmp/bilinote_backend.log`
- 前端日志: `/tmp/bilinote_frontend.log`
